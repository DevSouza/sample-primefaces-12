# Steps

1. Configure [Wildfly](./docs/configure-wildfly.md) with Postgres Module.
2. start infra with command `docker compose -f infra/compose.yml up -d`
3. Run project.

-   Plugin [EditorConfig](https://editorconfig.org/) for eclipse
